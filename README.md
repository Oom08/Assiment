Laundry Mart 🧺
Welcome to Laundry Mart, your one-stop solution for all laundry needs. This project is a simple, user-friendly website showcasing laundry services, pricing, and booking options.

📌 Project Overview
Laundry Mart provides:

Service details (washing, ironing, dry cleaning, stain removal, express service, etc.)

Transparent price list

Booking form for customers

Contact information with social media links

The goal of this project is to demonstrate a clean, responsive, and easy-to-navigate service website.

🚀 Features
Homepage with introduction and image banner

Services Section: Washing, Drying, Ironing, Folding, Dry Cleaning, Stain Removal, Express Services, Special Garment Care

Price List: Clear table with service costs

Booking Form: Collects customer details (Name, Address, Email, Phone)

Contact Section: Email, phone, Facebook, Instagram links

🛠️ Technologies Used
HTML5 for structure

CSS3 for styling

JavaScript (optional) for interactivity (if added later)

GitHub Pages for hosting
